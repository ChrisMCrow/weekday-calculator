# Weekday Calculator

#### Program return the day of the week - October 10, 2018

#### By **Brian Nelson** & **Chris Crow**

## Description

A simple program that has the user input a date and the day of the week is returned on the page.

### Specs    

Game and GamePlay

| Spec | Input | Output | Why |
